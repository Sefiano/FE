# Demo Data Pack

Drop these files into your frontend project's `src/data/` folder.

## Included files
- `types.ts`
- `inventorySeed.ts`
- `simulationSeed.ts`
- `worldModelSeed.ts`
- `decisionLogSeed.ts`
- `readinessSeed.ts`
- `index.ts`

## Example usage
```ts
import { inventorySeed, simulationSeed } from '@/data';
```

## Notes
- All data is hardcoded for demo use.
- Project context is seeded around **People & Places — The Med**.
- Types are intentionally permissive enough for UI work while still giving structure.
